The Complete Insurance Claim Management Playbook

Your comprehensive guide to maximizing every insurance claim. You carry the burden of proof—the insurer pays what you can prove. This Playbook shows you exactly how to secure, document, file, respond, negotiate, and close your claim — with AI support at every step.

How to use: Work section by section. Each area includes examples, decision trees, and an If → Then → Tool rail at the end so you always know the next best move and the right Claim Navigator tool to click.

1. Pre-Claim Preparation

Policy Documentation System

Action Required: Create a comprehensive binder: dec pages, riders/endorsements, premium proof (3y), all correspondence, prior claims, renewals, coverage changes.

Example: John's RCV endorsement on hand avoided an ACV-only opening offer and shaved 3 days off processing.

Digital Backup

  • Cloud folders: Year → Policy Type → Document Type
  • Password-protect sensitive docs; share with trusted contact
  • Quarterly updates

If → Then → Tool

Can't find full policy forms
Request certified copy citing state timeframe
·
Unsure if RCV or ACV applies
Run coverage decoder and flag endorsements
Home Inventory Documentation

Room-by-room list with model/serials, purchase date, cost & current replacement; photos (multi-angle), receipts, video walkthrough narration.

Case: Sarah's theft claim recovered $47k vs $28k initial due to receipts + appraisals + annual video.

If → Then → Tool

No receipts for older items
Use alt proofs: bank/CC statements, Amazon history, photos, social posts, warranty regs
High-value items (jewelry/art)
Obtain/refresh appraisals; check sub-limits and schedule
Pre-Loss Property Documentation

Monthly exterior photos (all sides/roofline), systems (HVAC/plumbing/electrical), flooring and finishes; date-stamp. Counters "pre-existing wear."

If → Then → Tool

Carrier claims pre-existing
Submit prior-condition photos and maintenance logs

2. Immediate Post-Loss Actions

Hour 1–2: Safety & Mitigation

Fire

  • Evacuate; wait for clearance
  • Board-up if needed; save invoices
  • Start safe-distance photos

Water

  • Shut off source; video active leak
  • Emergency extraction; dry & dehumidify
  • Move undamaged items
Critical: Policies require reasonable mitigation. Don't wait for adjuster permission—document and keep receipts.

If → Then → Tool

Home is uninhabitable
Request ALE advance and lodging authorization
Vendor requires insurer pre-approval
Proceed with emergency-only scope; notify carrier in writing with photos
Hour 2–6: Documentation

Wide → medium → close photos; serials; water/smoke lines; undamaged comparisons; written log of time/cause/weather/witnesses/actions.

If → Then → Tool

Lost critical receipts
Pull bank/CC PDFs; request store duplicates; annotate inventory
Hour 6–24: Initial Claim Contact

Prepare policy #, time/date, cause, report numbers, temp contact, emergency expenses. Use factual script.

If → Then → Tool

No acknowledgment in state timeframe
Send Delay Notice citing statute; request written status
·

3. Filing Your Claim

Proof of Loss & Packet

POL includes: policy/claim #, date/cause, full inventory, RCV/ACV calcs, doc references. Packet: 1-pg executive summary, room-by-room assessment, indexed photos, 3 estimates, code upgrades, ALE projections.

If → Then → Tool

Carrier asks for "more info" vaguely
Demand a specific, itemized list of missing documents
Inventory format rejected
Convert to carrier columns; keep your master file
Timeline Management

Days 1–3: report/mitigate/document · Days 4–7: inventory/estimates/submit · Weeks 2–3: inspection/estimate/review · Weeks 4–6: supplements/negotiation · >6 weeks: appraisal/attorney/DOI.

If → Then → Tool

Hard internal deadlines (POL due)
Calendar reminders + partial POL + "will supplement" notice
·

4. The Inspection Process

Pre-Inspection Preparation
  • Policy with highlights
  • Complete photo/video
  • 3+ estimates
  • Mitigation receipts
  • Item list + values
  • Code docs
  • Questions list
  • Witness present

If → Then → Tool

Adjuster limits scope/rooms
Request full walk; log refusal; follow with written recap
During the Inspection

Lead room-by-room; show hidden damage (under flooring, behind cabinets). Avoid speculation. Defer recorded statements until prepared.

If → Then → Tool

"Wear & tear" assertion
Counter with pre-loss photos, weather data, and contractor findings
Post-Inspection Follow-Up

Send same-day summary; attach photos; list "agreed items" vs "needs review"; note dates for next steps.

5. Estimate Review & Negotiation

Line-Item Analysis (Bathroom Example)

Adjuster estimate: $715 vs proper scope: $2,797 (moisture-resistant board, primer, 2-coat, plumbing wax ring, mitigation days, antimicrobial, moisture monitoring, subfloor, GFCI + inspection).

If → Then → Tool

Omitted code items
Add code citations + inspector notes + 3 contractor lines
Heavy/unreasonable depreciation
Challenge lifespan sources; request schedule and basis
Three-Estimate Strategy (Roof)

Use conservative/moderate/comprehensive bids to anchor scope; negotiate to mid point with full documentation.

If → Then → Tool

Carrier uses outdated unit costs
Submit current price indices + dated quotes
Supplements

Trigger: hidden damage during repairs, code adds, price changes, missed items. Provide discovery photos, expert notes, estimates.

6. Settlement Strategies

ACV vs RCV (Roof Example)

RCV $20k, depreciation $4k, deductible $2.5k → ACV check $13.5k; replace & submit completion docs to recover $4k holdback. Negotiate contractor to $19k to reduce out-of-pocket.

If → Then → Tool

Carrier delays payment after agreement
Send payment deadline reminder citing state statute
·
Negotiating Final Settlement

Use "show me the clause," line adjustments, written timelines, manager involvement. Don't sign immediately; request everything in writing.

Payment Structures
  • Single payment
  • ACV/RCV two-part
  • Milestones
  • Direct-to-vendor

7. Dispute Resolution

Internal Appeals

Appeal structure: undervaluation, overlooked damage, code requirements, policy interpretation with quotes. Request response in statutory timeframe.

If → Then → Tool

Denied as "gradual"
Submit plumber's sudden-failure report + neighbor affidavit + prior inspection
Appraisal

For amount-of-loss disputes; outline appraiser/umpire costs vs potential recovery; timing windows vary.

Public Adjuster & DOI Complaint

When to hire a PA (value/complexity); when to file a DOI complaint (delays, unfair practices, misrepresentation).

8. Post-Settlement Management

Managing Repairs

Vet contractors; milestone payments; lien waivers; warranty; supplement support clauses in contracts.

If → Then → Tool

Contractor requests >30% upfront
Negotiate milestone schedule; require lien waivers
RCV Recovery

Submit completion photos (before/during/after), invoices, proof of payment, permits/inspections, warranties to release depreciation.

Releases & Reservations

Avoid overbroad releases; preserve supplement rights; keep archive 7–10 years.

9. Special Circumstances

Total Loss

Emergency advance; rental; start contents inventory from memory; request police/fire reports; pull purchase histories.

If → Then → Tool

Mortgage company holds checks
Submit draw schedule + contractor contract + inspection proof
Business Interruption

Docs: 3y tax returns, 12-mo P&L, projections, seasonal adjustments, contracts/orders. Example: 45-day closure → lost revenue + continuing + extra expenses.

Catastrophe & Multiple Policies

Wind vs flood sequencing; FEMA registration; coordinate homeowners, NFIP, auto liability/umbrella; leverage causation engineering.

10. Advanced Strategies

Experts & Legal

When to hire engineers, meteorologists, hygienists, cause/origin; attorney fee models (contingency/hourly) based on claim size/complexity.

If → Then → Tool

Coverage denied on causation
Commission cause/origin + weather corroboration; submit rebuttal
Bad Faith Indicators

Delays, lowball without basis, misrep policy, refusal to provide docs, poor investigation, unnecessary demands, threats, altered reports.

Maximizing ALE & Claim File

Missed ALE: mileage, utilities, storage, pet boarding, meals, laundromat. Request full claim file under state statute.

11. Specific Peril Strategies

Lightning Strike

Hidden damage checklist: HVAC control boards, well pump controllers, landscape transformers, garage openers, doorbell transformers, irrigation controllers, pool/spa equipment, home theater wiring, cable/internet lines, security components, smart hubs, appliance boards.

If → Then → Tool

Adjuster only lists obvious electronics
Submit technician reports + itemized systems list with costs
Theft & Vandalism

Police report essentials; ownership proof via receipts, statements, photos, social posts, registrations, warranties, riders, appraisals.

Water (Burst Pipes) / Wind & Hail

Moisture mapping (wall cavities, subfloors, insulation, ducts); pre-winter occupancy evidence; weather docs; matching & uniformity with manufacturer discontinuation letters + code citations.

12. Property Type Specializations

Condominiums (Master vs HO-6)

Master: structure/common; HO-6: interior/contents; loss assessment coverage; betterments/improvements tracking.

If → Then → Tool

Master deductible passed to you
Invoke loss assessment coverage; cite bylaws and policy
Rental Property / Vacation Homes

Loss of rent: lease, rent roll, comps, property manager letter, occupancy rates. Occupancy proof for vacancy disputes: utilities, security logs, smart home data, inspections.

If → Then → Tool

Carrier alleges >60-day vacancy
Submit occupancy evidence pack; rebut vacancy exclusion

Final Thoughts & Best Practices

The 10 Commandments of Claim Management
  1. Document everything
  2. Never accept first offer
  3. Understand your policy
  4. Mitigate immediately
  5. Get multiple estimates
  6. Be persistent
  7. Know your rights (state rules)
  8. Get help when needed
  9. Stay organized
  10. Think long-term (supplements)
