Quick Example

POST /api/claim/roadmap
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "claim_type": "fire",
  "stage": 3,
  "jurisdiction": "CA"
}

Response:
{
  "status": "ok",
  "stages": 6,
  "current_stage": {
    "id": 3,
    "title": "Analyze & Compare",
    "tools": [...]
  }
}
