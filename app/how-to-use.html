How to Use This Site (Fast Start)

1) Create your case. Go to My Claims ‚Üí New Claim, select your claim type, state, and insurer, then upload any policy docs, letters, photos, and receipts.
2) Pick your path. Choose the workflow that matches your situation: New Filing, Denied, Low Offer, or No Response.
3) Follow the steps. Each section below has a short checklist. Complete tasks in order‚Äîyour Timeline and Deadlines will update automatically.
4) Generate documents. Use Document Generator to produce letters/forms, then send by the recommended channel (portal, email, certified mail).
5) Track & escalate. Log insurer calls/emails under My Claims. If stalled or denied, use Negotiation Scripts & Escalation and State-Specific Rights & Deadlines.

üü† Strategy & Guidance

Claim Playbook

Goal: A clear, customized plan from today to payout.

  1. Open Claim Playbook ‚Üí choose claim type + current status.
  2. Link your My Claims case to auto-pull policy limits, dates, and uploads.
  3. Complete Do This Now tasks.
  4. Stage This Week/This Month steps (evidence, estimates, letters).
  5. Add to Timeline to create due dates + reminders.
  6. Update as insurer responds.

Output: Personalized checklist + timeline entries.

Claim Timeline & Sequence Guide

Goal: Never miss a deadline; sequence actions correctly.

  1. Open Timeline ‚Üí confirm state + loss date.
  2. Import tasks from Playbook and Document Due Dates.
  3. Log insurer events (RFIs, inspections, offers, denials).
  4. Review computed deadlines (proof-of-loss, appeal windows, DOI timing).
  5. Optional: Calendar sync.
  6. Act on alerts (click red items to jump to the next action/letter).

Output: Live calendar with legal/contractual deadlines.

Insurance Company Tactics

Goal: Recognize and neutralize delay/denial tactics.

  1. Open Tactics Library ‚Üí pick what you're seeing.
  2. Click I'm seeing this to load counter-moves.
  3. Copy phone/email counter-script or generate a tailored letter.
  4. Log the event in My Claims ‚Üí Contacts & Notes.
  5. Escalate if repeated (supervisor ‚Üí manager ‚Üí DOI).

Output: Counter-scripts + escalation path + documentation template.

Maximize Your Claim

Goal: Increase payout with complete proof.

  1. Run Coverage & Gap Check (limits, deductible, ACV/RCV, exclusions).
  2. Upload proof in Evidence Organizer.
  3. Run Missing-Proof Scan by category (scope, causation, valuation).
  4. Get comps/estimates if needed (see Recommended Resources).
  5. Generate Supplemental Claim/Difference Letter.
  6. Log submission; watch Timeline for response window.

Output: Gap report + supplement letter + proof packet.

State-Specific Rights & Deadlines

Goal: Apply your state's protections.

  1. Select state to load statutes, rules, and DOI contacts.
  2. Enter loss/notice/denial dates to compute windows.
  3. Pick issue (delay, denial, underpayment, bad faith).
  4. Generate Right-to-Cure/Delay/Appeal/DOI Complaint letter.
  5. Send as instructed (certified mail/email/portal).
  6. Escalate after cure deadline if unmet.

Output: Statute-aware letters + DOI info + timing.

üü¢ Core Claim Tools

Advanced Tools

Goal: Deep analysis and QC.

  1. Choose tool (Policy Analyzer, Estimate Auditor, Settlement Comparison, Depreciation Check).
  2. Upload inputs (policy pages, estimates, letters).
  3. Run analysis ‚Üí see flags, citations, corrections.
  4. Generate targeted response letter/email.
  5. Save to My Claims and add follow-up reminder.

Output: Audit report + correction letter.

AI Response & Analysis Agent

Goal: Fast answers and cite-checked drafts.

  1. Open Agent and attach your case.
  2. Ask precisely (e.g., "Draft rebuttal citing Section D.3 for denied roof decking").
  3. Attach exhibits (photos/estimates).
  4. Verify citations.
  5. Choose tone (Professional/Direct/Firm) and refine.
  6. Export to Document Generator for final formatting/signature.

Output: Polished draft ready to send.

Claim Analysis Tools

Goal: Confident valuation and coverage mapping.

  1. Run Policy Breakdown (limits, sub-limits, ACV/RCV).
  2. Input loss data (sq ft/items/CPT codes as applicable).
  3. Review ACV vs. RCV, depreciation, likely settlement range.
  4. Compare to offer; note discrepancies.
  5. Generate Settlement Difference letter.

Output: Valuation model + comparison letter.

Situational Advisory

Goal: One-click playbook for your problem.

  1. Pick situation (first notice, stalled, denial, partial payment, low offer).
  2. Answer prompts (dates, claim #, received docs).
  3. Execute the solution stack (docs, scripts, deadlines).
  4. Mark done; it updates as responses arrive.

Output: Mini-playbook for your scenario.

Document Generator

Goal: Insurer-ready, court-ready documents.

  1. Choose template (Proof of Loss, RFI Response, Appeal, Supplemental Claim, Demand, DOI Complaint).
  2. Auto-fill from My Claims; complete remaining fields.
  3. Validate (missing data/dates/exhibits check).
  4. Sign & export (PDF/DOCX).
  5. Send & log delivery.
  6. Timeline adds response deadline automatically.

Output: Signed PDFs/DOCXs + logged delivery.

Evidence Organizer

Goal: Bulletproof documentation.

  1. Upload photos, invoices, receipts, estimates, reports.
  2. Auto-tag by area/date/type.
  3. Rename key files clearly.
  4. Link exhibits to letters.
  5. Export numbered Evidence Packet with table of contents.

Output: Organized evidence set linked to each letter.

Claim Document Library

Goal: Find exact template or sample.

  1. Filter by claim type, stage, or state.
  2. Preview sample vs. blank.
  3. Open in Document Generator to finalize.

Output: Right template, ready to fill.

Claim Documentation Guides

Goal: Do it right the first time.

  1. Pick document (e.g., Proof of Loss).
  2. Follow the checklist (field-by-field tips + pitfalls).
  3. Verify requirements (notary/witness/exhibits/delivery).
  4. Open matching template in Document Generator.

Output: Step-checked, compliant document.

Negotiation Scripts & Escalation

Goal: Professional communication + clear ladder.

  1. Pick scenario (low offer, no response, improper depreciation, denied line item).
  2. Copy phone/email script.
  3. Customize with facts, exhibits, deadlines.
  4. Deliver and log (who/when/what).
  5. Escalate to supervisor/manager/DOI on schedule.

Output: Conversation log + escalation letters & dates.

My Claims

Goal: Single source of truth.

  1. Create/select case (loss date, claim #, state, insurer).
  2. Upload policy, letters, estimates, photos; link contacts.
  3. Track offers & events (RFIs, inspections, calls).
  4. Watch deadlines (Timeline alerts).
  5. Export a complete claim file if needed.

Output: Organized case file with status, deadlines, history.
