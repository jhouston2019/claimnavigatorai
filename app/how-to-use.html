Quick Start

  1. Go to Situational Advisory, describe your scenario.
  2. Use the suggested buttons to open the correct Document Generator prefilled.
  3. Track deadlines with the State Rights and Trackers.
  4. Challenge offers using Estimate Comparison and Settlement Tool.

Everything opens in its own page—nothing hidden at the bottom.

Key Areas

  • AI Response Agent: paste insurer messages → draft replies.
  • Document Generator: create letters/forms quickly.
  • Document Library: find PDFs/templates stored in Supabase.
  • Guides: timeline & documentation best practices.
