## 🎓 USAGE EXAMPLES

### Python SDK v3.0

#### Policy Analysis
```python
from claimnavigator import ClaimNavigator

client = ClaimNavigator(api_key='your-key')

# Analyze policy with jurisdiction deadlines
result = client.analyze_policy_v3(
    claim_id='claim-123',
    policy_pdf_url='https://storage.example.com/policy.pdf'
)

# Access results
print(f"Policy Type: {result['policy_type']}")
print(f"Dwelling Limit: ${result['coverage']['dwelling_limit']}")
print(f"State: {result['jurisdiction']['deadlines']['state']}")
print(f"Deadline Warnings: {len(result['jurisdiction']['warnings'])}")

# Check for expired deadlines
for warning in result['jurisdiction']['warnings']:
    if warning['priority'] == 'critical':
        print(f"CRITICAL: {warning['title']}")
```

#### Estimate Analysis
```python
# Analyze estimates with advanced detection
result = client.analyze_estimates_v3(
    claim_id='claim-123',
    contractor_estimate_pdf_url='https://storage.example.com/contractor.pdf',
    carrier_estimate_pdf_url='https://storage.example.com/carrier.pdf',
    state='CA',
    carrier_name='State Farm'
)

# Access results
print(f"Base Recovery: ${result['exposure']['totalProjectedRecovery']}")
print(f"Total with Enforcement: ${result['exposure']['totalProjectedRecoveryWithEnforcement']}")

# Check pricing issues
pricing = result['enforcement']['pricingValidation']
print(f"Pricing Issues: {len(pricing['pricing_issues'])}")
print(f"Pricing Risk: ${pricing['report']['overall_assessment']['total_pricing_risk']}")

# Check depreciation abuses
depreciation = result['enforcement']['depreciationAbuse']
print(f"Depreciation Abuses: {depreciation['deterministic']['summary']['abuses_detected']}")
print(f"Depreciation Recovery: ${depreciation['deterministic']['summary']['total_recovery_potential']}")

# Check carrier tactics
tactics = result['enforcement']['carrierTactics']
print(f"Carrier Tactics: {tactics['summary']['total_tactics_detected']}")
print(f"Tactic Recovery: ${tactics['summary']['total_recovery_potential']}")

# Check for bad faith indicators
if 'systematic_underpayment' in tactics['summary']['tactic_types']:
    print("WARNING: Systematic underpayment pattern detected - consider attorney consultation")
```

## ⚠️ IMPORTANT DISCLAIMERS

### Pricing Validation
- Market data uses national averages with geographic adjustments
- Local market conditions may vary
- Specialized materials may not have market data
- **Recommendation:** Obtain local contractor quotes for validation

### Depreciation Analysis
- Schedules based on IRS Publication 946 and industry standards
- Actual depreciation may vary based on condition and maintenance
- **Recommendation:** Document actual condition with photos and expert opinion

### Carrier Tactics
- Pattern recognition identifies common tactics
- Not all tactics indicate bad faith
- **Recommendation:** Consult attorney if systematic patterns detected

### Jurisdiction Deadlines
- Deadlines are informational only
- State laws change - verify current requirements
- **Recommendation:** Consult licensed attorney for legal strategy

**REQUIRED for:**
- Claims >$50,000
- Bad faith indicators present
- Expired deadlines
- Systematic underpayment patterns
